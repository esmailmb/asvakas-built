Elevator Structural Consultation

Use this page when the project involves installation, replacement, modernization, shaft changes, pit work, support steel, or existing-building constraints that need building-side structural coordination in NYC.

Go to Contact FormBack to Elevator Service

Helpful intake items

  • Existing drawings and shaft information
  • New installation or modernization scope
  • Number of stops and building type
  • Any known filing or field constraints

What to prepare

The most useful starting package is the current scope description, existing building drawings if available, photos of the existing shaft or proposed location, and any indication that pit depth, overhead clearance, or support steel may change.

Common consultation topics

New installation feasibility

Where an elevator can fit in an existing building and what structure must change to receive it.

Modernization constraints

Whether the existing shaft, pit, and support condition still work for the revised equipment.

Filing coordination

How the building-side structural scope should be documented for the broader project team.
